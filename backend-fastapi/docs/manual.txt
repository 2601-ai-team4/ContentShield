# SNS Content Analyzer 사용 설명서

## 1. 개요
이 시스템은 YouTube 댓글과 SNS 콘텐츠의 악성 여부를 분석하는 AI 도구입니다.

## 2. 주요 기능
- **이중 모델 분석**: 빠른 분석(Groq Llama 3)과 정밀 분석(Google Gemini)을 결합하여 정확도를 높입니다.
- **실시간 대시보드**: 댓글의 유입 현황과 유해성 비율을 실시간으로 시각화합니다.
- **블랙리스트 관리**: 악성 댓글을 반복적으로 작성하는 사용자를 차단하고 관리할 수 있습니다.
- **RAG (문서 기반 답변)**: 사용자가 업로드한 문서를 바탕으로 AI가 질문에 답변합니다.

## 3. 설치 방법
1. 저장소를 클론합니다.
2. `backend-fastapi` 폴더에서 필요한 패키지를 설치합니다 (`pip install -r requirements.txt`).
3. `frontend-react` 폴더에서 `npm install`을 실행합니다.
4. 각 서버를 실행합니다.

## 4. 주의사항
- API 키가 올바르게 설정되어 있어야 합니다.
- 대량의 데이터를 분석할 때는 시간이 소요될 수 있습니다.
